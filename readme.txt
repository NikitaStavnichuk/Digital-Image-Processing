ЛАБ1:

При выполнении программы выводится меню с несколькими опциями на выбор пользователя:

1. Попиксельная сумма;
2. Произведение;
3. Среднее арифметическое;
4. Минимум;
5. Максимум;
6. Наложить на изображение маску;

При выборе каждой опции вводится полный путь первого и второго изображения. Изображения 
должны быть одного размера. В случае выбора 6 опции выбирается нужное изображение и 
форма маски:

1. Круг
2. Квадрат
3. Прямоугольник

После этого происходит обработка изображения по выбранной опции, успешно обработанное изображение предлагается
сохранить по указанной пользователем ссылке. Программа завершает работу.

ЛАБ2:

Сохранение функций с ЛАБ1 + добавлена функция градационного преобразования и расчёт гистограммы до градационного 
преобразования и после.
Функция (S = r ^ 2) задаётся статически.

При выборе опции 7 в меню

7. Градационные преобразования

вводится полный путь исходного изображения, полный путь куда сохранять гистограмму исходного изображения.
Далее обрабатывается исходное изображение, вводится путь куда сохранить обработанное исходное изображение.
Далее вводится путь сохранения гистограммы обработанного исходного изображения. Программа завершает работу.
